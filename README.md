# 📌 Project Overview

This project provides an interactive dashboard for exploring worldwide earthquake data. It combines **SQL-based analysis**, **Python data processing**, and **interactive visualizations** to uncover seismic patterns, magnitude trends, depth distribution, tsunami occurrences, and reporting quality.

The dashboard is designed to simulate a real-world business intelligence solution with executive metrics, analytical insights, and interactive SQL exploration.

# 📈 Key Insights

- Analyzed **114,000+** global earthquake records.
- Earthquake activity spans **230+ countries and regions**.
- Highest magnitude recorded: **8.8**
- 2025 recorded the strongest earthquake in the dataset.
- Alaska, Japan, Indonesia, and Russia remain among the most seismically active regions.
- The majority of earthquakes were classified as **Shallow**.
- Over **99%** of events were automatically reviewed.
- Most earthquake alerts were categorized as **Not Available**, with relatively few Red or Orange alerts.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/navinravan68-tech/Global-Seismic-Trends-Analysis.git
```

Move into the project

```bash
cd Global-Seismic-Trends-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/app.py
```
